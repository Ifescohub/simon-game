Press A Key to Start

How it woks !

This is a game of sequence. You can start the game by pressing any key on your keyboard

  • Firstly, the game shows the first colour in the sequence (e.g blue). The user clicks on the blue box.
  • Next, the game shows the next colour (e.g red ), the user has to remember the sequence is blue, red and click accordingly.
  • Next, the game shows the next colour (e.g green), the user follows the sequence and click blue, red , green and so on and so forth.
  • If the user messes up the sequence, then the game ends. The user must press any key on the keyboard to start again.

Phone user can click the input area (white space) below the boxes to activate his/her keyboard.
